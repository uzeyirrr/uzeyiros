ÜzeyirOS v1.1.0
===============

Bu ISO dosyası ÜzeyirOS işletim sistemini içerir.

Kullanım:
- QEMU ile: qemu-system-x86_64 -cdrom uzeyiros.iso
- Gerçek donanımda: CD/DVD'den boot

Özellikler:
- Interactive GUI
- Real-time shell
- Keyboard input support
- Multi-processor support

Geliştirici: Üzeyir İsmail Bahtiyar
GitHub: https://github.com/uzeyirrr/uzeyiros.git
